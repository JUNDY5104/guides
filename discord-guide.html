Discordとは何か（3分で理解）

Discord（ディスコード）は、テキスト・音声・ビデオ・画面共有がすべて無料でできるコミュニケーションツールです。2015年にアメリカで誕生し、2026年現在は月間1億5,000万人以上が使っています。

もともとはゲームをしながら仲間と話すために作られましたが、今ではゲームに限らず、勉強グループ・仕事チーム・趣味のコミュニティ・AIツールの公式コミュニティなど、あらゆる場面で使われています。

3つの例えで一発理解
🏠
サーバー
= 建物全体。「〇〇部」「〇〇コミュニティ」という集まりの単位
🚪
チャンネル
= 建物の中の部屋。「雑談部屋」「お知らせ部屋」「通話部屋」など
🎭
ロール
= 役職・バッジ。「管理者」「メンバー」「VIP」など役割を分ける仕組み

LINEやSlackとどう違うの？

比較DiscordLINESlack
主な用途コミュニティ・友人グループ個人間の連絡仕事チーム
音声通話無料・高音質有料通話あり有料プラン
グループの規模数人〜数万人まで対応主に小規模主にチーム規模
画面共有無料で使えるなし有料プランのみ
ボット（自動化）豊富・無料限定的あり
料金基本無料基本無料無料〜有料
まず知っておくこと Discordは「難しい」という印象があるけれど、実は無料で機能が充実しているツールです。用語が独特なので最初は戸惑いますが、このガイドを読めば全部わかります。

アカウント作成・初期設定

アカウント作成は5分以内で完了します。メールアドレスだけあれば始められます。

アカウント作成の手順

  1. discord.com にアクセスして「Discordを開く」または「Discordをダウンロード」をクリックする（PC版アプリのインストールが推奨。ブラウザでも使える）
  2. 「登録する」をクリックしてメールアドレス・ユーザー名・パスワード・生年月日を入力する
  3. 登録したメールに確認メールが届くのでリンクをクリックして認証する
  4. ホーム画面が表示されれば登録完了。すぐに使い始められる

最初にやっておくべき3つの設定

設定場所やること
アイコン・プロフィール設定左下の自分のアイコン→「プロフィールを編集」顔写真またはアイコン画像を設定する。ひひひ一言コメントも入れると親しみやすい
マイク・スピーカーのテスト左下の歯車→「音声・ビデオ」通話前に必ず確認。「マイクのテスト」ボタンで自分の声が入っているか確認する
通知の設定左下の歯車→「通知」最初は全通知がオンになっているので、不要な通知をオフにする（詳しくはSection 09）
ユーザー名について 2023年のアップデートから、Discordのユーザー名は「username」形式（数字のタグ#0000が不要）になりました。自分の名前として使いたい英数字の名前を設定してください。後から変更も可能です。

「サーバー」って何？参加の仕方

サーバーとは、Discordの中にある「コミュニティ空間」のことです。LINEのグループに近いですが、もっと大きくて機能が充実しています。

これで完璧にわかる例え

サーバー = マンション全体。その中にいくつもの「チャンネル（部屋）」があります。マンションに入居する（サーバーに参加する）ことで、そのマンション内のすべての部屋に出入りできます。

サーバーに参加する3つの方法

方法手順使う場面
招待リンクから参加管理者から届いたURLをクリックするだけ。「サーバーに参加」ボタンを押して完了友達や知人のサーバーに招待された時
招待コードを入力左端の「+」ボタン→「サーバーに参加する」→招待コード（英数字）を入力SNSでコードだけ教えてもらった時
公開サーバーを検索左端の下にある「コンパス」アイコンをクリック→サーバーを検索して参加自分から興味あるコミュニティを探したい時

サーバーに参加したら最初にすること

  1. 「#ルール」または「#rules」チャンネルを読む： ほとんどのサーバーにルールチャンネルがある。禁止事項や注意事項が書かれているので必ず確認する
  2. 「#お知らせ」「#自己紹介」チャンネルを確認する： サーバーの目的・最近の情報・自己紹介のフォーマットが書かれていることが多い
  3. 不要な通知をミュートする： チャンネル名を右クリック→「通知設定」→「ミュート」で特定のチャンネルの通知を消せる
サーバーを退出するには 左のサーバーアイコンを右クリック→「サーバーから退出する」で退出できます。退出してもメッセージ履歴には影響しません。間違って退出しても、再度招待リンクをもらえば再参加できます。

「チャンネル」って何？種類と使い方

チャンネルとは、サーバー内にある「話題ごとの部屋」です。同じサーバーの中にたくさんの部屋があり、目的ごとに使い分けます。

# テキストチャンネル
= 文字でやり取りする部屋
メッセージ・画像・リンクなどを送る。一番基本的なチャンネル。「#雑談」「#質問」「#お知らせ」など
🔊 ボイスチャンネル
= 音声で話す部屋
クリックするだけで入室して通話開始。テキストも同時に送れる。「General」「作業部屋」など
📢 アナウンスチャンネル
= 管理者からのお知らせ専用部屋
メンバーは読むだけで書き込めないことが多い。重要なお知らせが流れてくる
📁 フォーラムチャンネル
= 質問・話題ごとにスレッドが立つ部屋
話題ごとにスレッド（独立した会話）が作られる。質問が多いコミュニティでよく使われる

カテゴリとは

チャンネルをまとめるフォルダのようなものです。「お知らせ」「雑談」「ゲーム部屋」などのカテゴリでチャンネルをグループ化して見やすくします。カテゴリ名をクリックすると中のチャンネルを折りたたんで隠すこともできます。

チャンネルが多すぎて困った時

チャンネルが多すぎると感じたら、チャンネル名を右クリック→「チャンネルをミュート」を活用しましょう。ミュートしたチャンネルは通知が来なくなります。全部読む必要はありません。自分に関係あるチャンネルだけを使えばOKです。

メッセージの送り方・基本操作

Discordのテキスト操作はLINEに近いですが、いくつか便利な機能があります。

メッセージの基本操作

操作やり方
メッセージを送る入力欄に文字を入力してEnterキー（改行はShift+Enter）
メッセージを編集する自分のメッセージにカーソルを当てて鉛筆アイコン→テキストを変更してEnter
メッセージを削除する自分のメッセージにカーソルを当てて「・・・」→「メッセージを削除」
特定のメッセージに返信するメッセージにカーソルを当てて矢印（返信）アイコンをクリック
リアクション（絵文字）をつけるメッセージにカーソルを当てて笑顔アイコン→絵文字を選択
メッセージをピン留めする「・・・」→「メッセージをピン留めする」（管理者権限が必要なことも）

文字の装飾（マークダウン）

【テキストを装飾する書き方】
**太字**      → 太字になる
*斜体*        → 斜体になる
__下線__       → 下線が引かれる
~~取り消し線~~ → 取り消し線が引かれる
`コード`       → 等幅フォントのコード表示
```
複数行の
コードブロック
```              → コードブロック（プログラムを貼る時に使う）

> 引用文        → メッセージの先頭に縦線が入った引用スタイル
||ネタバレ||    → クリックするまで隠れるテキスト（ネタバレ防止）

メンションの使い方

@ユーザー名     → 特定の人に通知が届く（相手が気づけるように使う）
@everyone      → サーバーの全メンバーに通知（乱用注意！）
@here          → 現在オンラインのメンバー全員に通知
@ロール名       → 特定のロールを持つメンバー全員に通知
#チャンネル名   → チャンネルへのリンクを貼る

ボイスチャット・ビデオ通話の使い方

Discordのボイスチャットは入退室がとても簡単です。ボイスチャンネルをクリックするだけで通話が始まります。ZoomやLINE通話と違って「発信する・呼び出す」という操作が不要です。

ボイスチャットの基本操作

  1. 入室： チャンネルリストのボイスチャンネル（🔊マーク）をクリックするだけで即接続
  2. マイクOFF（ミュート）： 左下のマイクアイコンをクリック。赤くなればミュート状態。Discordを使っていない間は必ずミュートを推奨
  3. スピーカーOFF（耳マーク）： 左下の耳アイコンをクリックで相手の声が聞こえなくなる
  4. 退室： 左下の「通話を切断」（電話に斜線のアイコン）をクリック

ビデオ通話・画面共有

機能操作方法メモ
ビデオ通話（カメラON）ボイスチャンネルに接続後、下部の「カメラ」アイコンをクリック事前にカメラの設定で動作確認を推奨
画面共有ボイスチャンネルに接続後、「画面」アイコンをクリック→共有したいウィンドウを選択ゲーム配信・作業配信・プレゼンに使える
ゴーライブ（配信）ゲーム中に「ゴーライブ」ボタンが表示される→クリックでサーバーに配信開始同サーバーのメンバーが視聴できる
「誰も話していないのに通話中になっている」はよくある失敗 ボイスチャンネルは退室操作をするまでずっと接続中のままです。使い終わったら必ず「通話を切断」ボタンを押して退室してください。接続しっぱなしで気づかないことが多いので注意しましょう。

08

フレンド登録・DM（個別メッセージ）

Discordでは特定の相手と1対1でメッセージや通話ができます。これをDM（ダイレクトメッセージ）といいます。DMを送るには基本的にフレンド登録が必要です。

フレンド登録の方法

  1. 画面左上のDiscordロゴ（またはHomeアイコン）をクリックしてホーム画面を開く
  2. 「フレンド」タブをクリックして「フレンドを追加」ボタンを押す
  3. 相手のユーザー名を入力して「フレンド申請を送る」をクリックする
  4. 相手が承認するとフレンドリストに追加される
サーバー内のメンバーにDMを送る簡単な方法 サーバーのメンバーリストまたはチャット内のユーザー名をクリック→「メッセージを送る」でDMを送れます。フレンド登録なしでも送ることができます（相手側の設定によっては受信できない場合もあります）。

グループDM（3人以上のプライベート通話）

フレンドとのDM画面の右上にある「グループDMに追加」ボタンから最大9人までのグループDMを作れます。サーバーを作るほどでもない少人数のやり取りに便利です。

09

通知設定・うるさい通知を消す方法

Discordは初期設定のままだと通知が多すぎて大変です。通知の設定を整えることがDiscordを快適に使うための最重要ポイントです。

通知を整える3段階の設定

設定場所やること操作方法
アプリ全体デスクトップ通知をオフにする（または必要なものだけに絞る）左下の歯車→「通知」→「デスクトップ通知を有効にする」をオフ
サーバー単位特定のサーバー全体の通知をミュートするサーバーアイコンを右クリック→「通知設定」→「ミュート」
チャンネル単位特定のチャンネルだけ通知をオフにするチャンネル名を右クリック→「通知設定」→「ミュート」
 
【通知設定の3つのモード】
「すべてのメッセージ」 
→ チャンネルに書き込みがあるたびに通知（賑やかなサーバーには不向き）

「@メンション、@roles、@everyone のみ」 ← これが一番おすすめ
→ 自分への呼びかけがあった時だけ通知。静かで快適

「何も通知しない」
→ 一切通知しない。あとで読み返す人向け
おすすめ設定：メンション時のみに絞る 全サーバーの通知を「@メンション時のみ」に設定するのが最もバランスが良いです。サーバーアイコンを右クリック→「通知設定」→「@メンション、@roles、@everyone のみ」を選択してください。

自分のサーバーを作る方法

Discordのサーバーは無料で何個でも作れます。友達グループ・仕事チーム・コミュニティなど目的に応じて作ってみましょう。

サーバー作成の手順

  1. 左端のサーバーリスト一番下の「+」ボタンをクリックする
  2. 「オリジナルの作成」を選択する（テンプレートを使うと最初からチャンネルが整理された状態で作れる）
  3. サーバーの用途を選択する（「友達グループ向け」「クラブやコミュニティ向け」など）
  4. サーバー名・アイコン画像を設定して「完成」をクリック
  5. デフォルトで「一般」テキストチャンネルと「一般」ボイスチャンネルが作られた状態でスタート

友達を招待する方法

  1. サーバー名をクリック→「招待リンクを作成する」を選択する
  2. 招待リンクが生成されるのでコピーして相手に送る
  3. リンクの有効期限（7日間・24時間・永久など）を設定できる。コミュニティ向けには「永久」が便利

最低限作っておくべきチャンネル構成

【小規模（友達グループ・5〜20人向け）の基本構成】
カテゴリ：INFO
  #ルール（管理者のみ書き込み可）
  #自己紹介

カテゴリ：TALK
  #雑談
  #質問

カテゴリ：VOICE
  🔊 雑談部屋
  🔊 作業部屋

【ポイント】
→ チャンネルは少なめからスタートする（多すぎると管理が大変）
→ 後から追加するのは簡単なので、最初はシンプルに

ロール（役職）・権限の設定

ロールとは、サーバーのメンバーに付ける「役職・バッジ」のことです。「管理者」「一般メンバー」「VIP」など、役割ごとに異なる権限を設定できます。

ロールの例え

会社の「部長」「社員」「アルバイト」のような役職です。部長は全部屋に入れるけど、アルバイトは一部の部屋には入れない、というような権限の差をDiscordでも設定できます。

ロールの設定方法

  1. サーバー名をクリック→「サーバー設定」を選択する
  2. 左メニューの「ロール」をクリックして「ロールを作成」ボタンを押す
  3. ロール名・色・権限（チャンネルの閲覧可否・管理権限など）を設定する
  4. 「メンバー」タブでサーバー参加者にロールを割り当てる

よく使うロールの設定例

ロール名権限の設定例使い方
@管理者全権限ありサーバーのオーナー・信頼できる運営メンバーに付与
@モデレーターメッセージ削除・メンバーのキック可荒らし対策を担う人に付与
@メンバー通常チャンネルの閲覧・発言可一般参加者全員に付与
@VIP限定チャンネルへのアクセス可特別なメンバーや有料メンバーに付与

有料プラン「Nitro」とは

Discordの基本機能はすべて無料ですが、Nitro（ナイトロ）という有料プランに加入するとさらに便利になります。ただし、普通に使うだけなら無料で十分です。

Nitroプランの比較（2026年版）

プラン料金（月額）主な特典
無料0円基本的な機能すべて。通常利用に十分
Nitro Basic約350円/月カスタム絵文字の使用・プロフィールバナー・アニメーションアバター
Nitro約1,100円/月Basic全機能＋ファイルアップロード上限500MB・サーバーブースト2個・動画の4K配信・カスタムサウンドボード

サーバーブーストとは

サーバーブーストは、特定のサーバーに「ブースト」を付与する機能です。ブーストが集まるとサーバー全体の機能が強化されます。

  • レベル1（2ブースト）： サーバーのカスタム絵文字50個追加・音質向上
  • レベル2（7ブースト）： サーバーバナー・招待URL設定・絵文字さらに追加
  • レベル3（14ブースト）： カスタムドメイン・最高音質・絵文字さらに拡張
結論：Nitroは不要な人の方が多い テキストチャット・音声通話・画面共有・サーバー作成など主要機能はすべて無料で使えます。カスタム絵文字をたくさん使いたい・高画質で配信したい・お気に入りのサーバーを応援したい場合にNitroを検討してください。

よくある困りごと・FAQ

サーバーに参加したのに何も見えません。チャンネルがありません。
ロール（役職）が割り当てられていないため、チャンネルが見えない状態の可能性があります。多くのサーバーでは参加後に「#ルール」を読んで絵文字をクリックするとロールが付与されてチャンネルが見えるようになる仕組みになっています。まず「#ルール」「#自己紹介」などのチャンネルを確認してください。
マイクが反応しません。声が相手に届きません。
左下の歯車（設定）→「音声・ビデオ」→「入力デバイス」で正しいマイクが選択されているか確認してください。「入力感度」の設定で「入力感度を自動的に判断する」をオフにして、スライダーを手動で調整すると改善する場合があります。また「音声設定をリセット」ボタンで初期化する方法も有効です。
通知が多すぎてうるさいです。
各サーバーのアイコンを右クリック→「通知設定」→「@メンション、@roles、@everyone のみ」に変更してください。これで自分宛てのメンションがあった時だけ通知が届きます。特に賑やかなサーバーは「ミュート」にするのも有効です。
誤って送ったメッセージを消したい。
自分が送ったメッセージにカーソルを当てると右上に「・・・」が表示されます。そこをクリックして「メッセージを削除」を選択してください。削除したメッセージは元に戻せません。また自分のメッセージしか削除できません（管理者は他人のメッセージも削除できます）。
スマホとPCで同じアカウントを使えますか？
はい、同じアカウントでスマホ・PC・ブラウザのすべてから使えます。サーバーへの参加状況・メッセージ履歴はすべてのデバイスで同期されます。
Discordで知らない人から突然DMが来ます。どう対処すればいいですか？
設定→「プライバシー・安全」→「サーバーメンバーからのダイレクトメッセージを許可する」をオフにすることで、フレンド以外からのDMをブロックできます。また怪しいリンクが送られてきた場合は絶対にクリックしないでください。スパムや詐欺の可能性があります。
アカウントを削除したい。
設定→「マイアカウント」→「アカウントを無効化する」（一時的な停止）または「アカウントを削除する」（完全削除）が選べます。完全削除は14日間の猶予期間後に実行されます。削除すると送ったメッセージの表示が「削除されたユーザー」になりますが、メッセージ内容は残ります（サーバー管理者のみ見える形で残ることがあります）。
