アカウント作成・初期設定

アカウント作成の手順

  1. chatwork.com にアクセスして「無料で始める」をクリックする
  2. メールアドレスを入力して「無料アカウントを作成」をクリックする
  3. 登録メールに届いた確認リンクをクリックして認証する
  4. 氏名・会社名・パスワードを設定してアカウント作成完了
  5. PC版アプリをダウンロードするか、ブラウザ版（chatwork.com）でそのまま使い始める

最初にやっておくべき設定

設定項目場所内容
プロフィール設定右上アイコン→「プロフィール編集」氏名・プロフィール画像・部署・電話番号など。社内外の相手に見えるので丁寧に設定する
通知設定設定→「通知」メール通知・デスクトップ通知の受け取り方を設定。最初は全通知オンで様子を見る
二段階認証設定→「セキュリティ」不正ログイン防止のために必ず設定する（詳しくはSection 12）
マイチャット活用チャット一覧「マイチャット」自分だけが見られるメモ帳として活用。下書きやアイデアメモに使う

グループチャットの使い方

Chatworkの中心機能がグループチャットです。部署・プロジェクト・取引先など目的ごとにチャットルームを作成して会話を整理します。

グループチャットの作成

  1. 左上の「+」ボタン（またはチャット一覧上部の新規作成アイコン）をクリックする
  2. 「グループチャットを作成」を選択する
  3. チャット名・アイコンを設定する（後から変更可）
  4. 参加させたいメンバーをコンタクトリストから選択する
  5. 「作成」をクリックして完了。すぐにメッセージを送り始められる

グループチャットの運用ポイント

【チャット名の付け方】
目的・人数・期間が一目でわかる名前にする

例：
「営業部_全体」          → 部署の常設チャット
「★ProjectA_開発チーム」 → ★をつけて重要チャットを上位表示
「【2026年3月】キャンペーン企画」→ 期間限定プロジェクト用
「山田さん_株式会社〇〇」  → 取引先との1:1

【ピン留め機能の活用】
よく使うチャットを右クリック→「ピン留め」で
チャット一覧の上部に固定表示できる

【チャットの退出・アーカイブ】
プロジェクト完了後は「チャットを退出」または
「アーカイブ」で整理する

タスク管理機能の使い方

Chatworkのタスク機能は、メッセージと連動して作業の依頼・受領・完了を管理できる機能です。メールで言う「〇〇お願いします」という依頼を、タスクとして追跡・管理できます。

タスク一覧（未完了）
企画書の修正・再提出（担当：田中）
3/20 期限切れ
請求書の確認と承認
3/25
来月のMTGスケジュール調整
3/31
✓
サーバーの設定確認
完了

タスクの作成と割り当て

  1. チャット画面右上の「タスク」タブをクリックする（または右側のタスクエリアから）
  2. 「タスクを追加」ボタンをクリックしてタスク内容を入力する
  3. 担当者（自分または他のメンバー）・期限を設定する
  4. 「追加」をクリックすると、担当者に通知が届く
  5. タスク完了時はチェックボタンをクリック。依頼者に完了通知が届く

タスク機能の活用ポイント

【タスクを使うべき場面】
「〇〇してください」という依頼
「確認お願いします」という確認依頼
「期限：〇月〇日まで」という締め切りがある作業

【使わなくていい場面】
雑談・情報共有・質問（普通にメッセージで十分）

【よくある使い方の失敗】
× チャットで「よろしくお願いします」とだけ送る
  → 流れていって誰も対応しない

○ タスクに登録して担当者・期限を明確にする
  → 完了まで追跡できる・抜け漏れが防げる

【注意点】
- 自分が依頼した以外の他のメンバーのタスクは見えない
- リマインド機能はないため、期限管理は自分で行う

ファイル共有・管理

Chatworkではチャット内でファイルを直接共有・管理できます。メールに添付するよりも簡単で、ファイルのやり取り履歴もチャットに残るため、後から確認しやすいのが特徴です。

ファイルのアップロード方法

  1. チャット入力エリアのクリップアイコンをクリックしてファイルを選択する（またはドラッグ＆ドロップ）
  2. 必要であればコメントを入力して送信する
  3. アップロードされたファイルは「ファイル」タブで一覧確認できる

ファイル管理の仕様

項目フリープランビジネス以上
ストレージ容量組織全体で5GB組織全体で10GB以上
1ファイルの上限5MB5MB（同様）
ファイル検索制限あり詳細検索可
ファイルの閲覧直近40日以内全期間
機密ファイルの取り扱い注意 Chatworkで共有したファイルはチャットのメンバー全員が閲覧できます。機密性の高い書類をグループチャットで共有する際は、参加メンバーの範囲を確認してから送信してください。

ビデオ通話・音声通話

Chatworkにはビデオ通話・音声通話機能が搭載されており、外部ツールなしにオンライン会議を開催できます。

通話の開始方法

  1. 通話したいチャットを開く
  2. チャット右上のビデオカメラアイコンをクリックする
  3. 「通話を開始する」をクリックすると通話ルームが作成される
  4. チャット内に通話リンクが投稿されるので、参加者はそこをクリックして入室する

通話機能の仕様比較

機能フリープランビジネス以上
1対1の通話可可
複数人のビデオ会議不可（1対1のみ）最大14人
画面共有可可
複数人での会議はビジネスプラン以上が必要 フリープランでは1対1の通話しかできません。3人以上でのビデオ会議を行う場合は、ビジネスプラン以上への移行が必要です。または外部ツール（Zoom・Google Meet）とChatworkを組み合わせる方法もあります。

コンタクト追加・管理

Chatworkでは、他のユーザーとチャットするには事前にコンタクト追加が必要です（またはグループチャット経由での接触）。

コンタクトの追加方法

方法手順使う場面
メールアドレスで検索コンタクト追加→メールアドレスを入力→申請を送る→相手が承認相手のメールアドレスを知っている時
Chatwork IDで検索コンタクト追加→Chatwork IDを入力→申請相手のIDを教えてもらった時
グループチャット経由共通のグループチャット内のメンバー名をクリック→コンタクト申請共通のチャットに既にいる相手
知らない相手からのコンタクト申請は必ず確認してください
  • 相手のメールアドレスさえ知っていれば誰でもコンタクト申請を送れる仕様です
  • 申請者の名前・プロフィール写真が本物に見えても、なりすましの可能性があります
  • 既に同じ人のアカウントをコンタクト登録済みでないか必ず確認してください
  • 不審な申請は承認せずに「拒否」してください

通知設定の最適化

通知の種類設定方法おすすめ
デスクトップ通知設定→「通知」→デスクトップ通知をON/OFF業務時間中はON推奨

マイチャットの活用方法

マイチャットは自分だけが見られるプライベートなチャットです。以下のような使い方が便利です。

  • メモ帳として： 思いついたアイデア・覚えておきたいことを書き留める
  • 下書きチェックとして： 送信前のメッセージをここで書いて確認してから本番チャットに貼り付ける
  • ToDo管理として： 自分だけのタスクリストをマイチャットのタスク機能で管理する
  • ファイル置き場として： よく使うファイルをアップロードして手軽にアクセスする

予約送信機能（有料プランのみ）

2024年11月に追加された機能です。メッセージ入力後に時計アイコンをクリックして日時を指定すると、設定した日時に自動でメッセージが送信されます。早朝に作業した内容を営業時間内に送りたい場合などに便利です。

なりすまし詐欺の手口と対策【重要・2026年最新】

公式緊急注意喚起（2026年1月・kubell発表）
  • 実在する企業の代表者・社員を装い、口座情報の提供や現金の振り込みを要求するなりすまし詐欺が急増しています
  • 実在する人物の氏名・プロフィール画像を無断使用した悪質なアカウントが確認されています
  • 2026年1月20日に通報窓口が新設され、警察との連携も開始されています

主な手口（3パターン）

手口具体的な内容狙い
振込詐欺型社長・上司を装って「至急この口座に振り込んでください」「振込先が変わりました」とメッセージを送る不正送金
フィッシング型「システムのアップデートが必要です」「セキュリティ確認のためログインしてください」という偽サイトへのリンクを送るアカウント乗っ取り
多段階誘導型Chatworkで接触後、LINEグループへ誘導。「他の人には知らせないで」と孤立させてから要求を繰り返す金銭・情報搾取

なぜChatworkが狙われやすいのか

  • 相手のメールアドレスを知っているだけで申請できる： 承認ハードルが低く、名前と顔写真を本物そっくりにすると騙されやすい
  • チャットは「信頼の場」という心理： メールより警戒心が薄れ、上司・取引先からと思うと反射的に従ってしまいがち
  • 「至急」という言葉で判断力を奪う： 急かされると確認する余裕がなくなる

絶対に覚えておくべき鉄則

これが来たら100%詐欺です
  • チャット上で突然「口座に振り込んでください」「口座番号を教えてください」と言われる
  • 「至急」「緊急」「今だけ」「他の人には内緒で」というキーワードが入っている
  • 普段やり取りしていない相手から突然コンタクト申請が来て、すぐに金銭の話になる
  • 既にコンタクト登録している同じ人から「新しいアカウントです」と申請が来る
なりすまし詐欺を防ぐ今すぐできる対策チェックリスト
  • 知らないアカウントからのコンタクト申請は、既存コンタクトと同名がないか確認してから承認する
  • コンタクト申請者のChatwork IDとプロフィールを、知り合いの正規アカウントと比較確認する
  • チャット上での振込・口座情報の要求は、必ず電話または対面で本人確認する
  • 「他の人には言わないで」という指示が来たら詐欺と判断して応じない
  • 二段階認証を設定してアカウント乗っ取りを防ぐ
  • 不審なURLは絶対にクリックしない
  • 被害を受けたまたは受けそうになった場合、警察または消費生活センターに相談する

なりすましアカウントを発見したら

  1. やり取りの記録（スクリーンショット）を保存する
  2. 不審なアカウントとのコンタクトを削除する（コンタクトリスト→右クリック→「コンタクトを削除」）
  3. Chatwork内の「ご意見・ご要望」からkubellに報告する（2026年1月〜通報窓口が新設）
  4. 金銭的な被害が発生した場合は最寄りの警察署または消費生活センターに相談する
誤って承認してしまった場合 不正アカウントのコンタクト申請を誤って承認してしまった場合は、「コンタクトの削除」と「グループチャットからの削除」を行ってください。金銭的要求には絶対に応じず、すぐに警察に相談してください。

セキュリティ設定

二段階認証の設定（全ユーザー必須）

  1. 右上のアイコン→「設定」→「セキュリティ」をクリックする
  2. 「二段階認証を設定する」をクリックする
  3. 認証アプリ（Google AuthenticatorまたはMicrosoft Authenticator推奨）をスマートフォンにインストールする
  4. QRコードをアプリで読み取り、表示される6桁のコードを入力して設定完了
二段階認証は絶対に設定してください 二段階認証を設定するとパスワードが漏洩してもログインできません。なりすまし詐欺対策の最も基本的かつ効果的な対策です。特に管理者アカウントは必須です。

よくあるトラブル・FAQ

メッセージが「直近40日以前」で見られなくなりました。
フリープランの仕様です。40日以前のメッセージはフリープランでは閲覧できません。過去のメッセージへのアクセスが必要な場合はビジネスプランへのアップグレードが必要です。なお、メッセージ自体は消えているわけではなく、有料プランにアップグレードすれば遡って閲覧できます。
知らない人からコンタクト申請が来ました。承認してもいいですか？
慎重に確認してください。相手のプロフィールを見て、すでに同じ名前の人をコンタクト登録していないか確認してください。知人・取引先の場合は別の手段（電話・メール）で本人確認してから承認することをおすすめします。不審な場合は拒否してください。2026年1月以降、なりすまし詐欺が急増しているため特に注意が必要です。
送ったメッセージを取り消せますか？
Chatworkでは送信したメッセージを「編集」または「削除」できます。メッセージにカーソルを当てると表示される「・・・」から選択してください。ただし、相手がすでに読んでいた場合、内容を見られている可能性はあります。また管理者は他のメンバーのメッセージも削除できます。
退職した社員のアカウントはどうすればいいですか？
管理者が「ユーザー管理」からそのユーザーを「無効化」または「削除」できます（ビジネスプラン以上）。無効化するとログインできなくなります。退職者のアカウントを放置するとセキュリティリスクになるため、退職時には必ず対応してください。
スマホとPCで同じアカウントを使えますか？
はい、同じアカウントでスマホアプリ・PCアプリ・ブラウザ版の全てから使えます。メッセージ・タスク・ファイルはすべてのデバイスで同期されます。
グループチャットを誤って退出してしまいました。
退出した場合、自分でチャットに戻ることはできません。チャットのオーナーまたは管理者に再招待してもらう必要があります。なお、退出後もそれ以前のメッセージ閲覧履歴には影響しません。
