01

Chatworkとは

Chatwork（チャットワーク）は、株式会社Chatworkが提供する国内最大級のビジネスチャットツールです。2011年のサービス開始以来、中小企業を中心に450万社以上が導入しており、チャット・タスク管理・ファイル共有・ビデオ通話をひとつのツールで完結できる点が最大の特長です。

Chatworkでできること

機能概要
グループチャット部署・プロジェクト・取引先ごとにチャットルームを作成して会話を整理できる
ダイレクトメッセージ1対1の個別チャット。社内外を問わず気軽に連絡できる
タスク管理チャット内のメッセージをワンクリックでタスク化。担当者・期日を設定して管理できる
ファイル共有画像・PDFなどのファイルをチャット内に添付して共有。プランごとに保存容量が異なる
ビデオ・音声通話画面共有機能付きのビデオ通話をChatwork内から直接発信できる
コンタクト管理社内メンバーと社外（取引先・クライアント）を分けてコンタクトを管理できる

他ツールとの違い

比較項目ChatworkSlackTeams
主なユーザー層中小企業・個人事業主スタートアップ・IT系大企業・Microsoft環境
社外との連携メール不要でコンタクト申請だけで連携可能ゲストアカウント制限ありゲストアカウント設定が複雑
タスク管理標準機能として内蔵別途アドオンが必要Plannerとの連携が必要
日本語対応完全日本語対応・国内サポート充実英語UI中心日本語対応だが設定が複雑
無料プランあり（機能制限つき）あり（メッセージ保存数制限）あり（Microsoft 365前提）
Chatworkが選ばれる理由

ITリテラシーが高くない従業員でも使いやすいシンプルな画面設計と、社外の取引先・クライアントともコンタクト申請だけで簡単に連携できる点が、中小企業に支持される大きな理由です。

無料・有料プランの違い

Chatworkは無料プランから利用を始められますが、業務での本格活用には有料プランへのアップグレードが推奨されます。各プランの主な違いを確認しましょう。

  • ファイルストレージ 5GB

アカウント作成と初期設定

Chatworkのアカウントはメールアドレスがあれば無料で作成できます。以下の手順で5分程度で完了します。

アカウント作成手順

  1. Chatwork公式サイト（chatwork.com）にアクセスし「無料で始める」をクリックする
  2. メールアドレスを入力して「次へ」をクリックする
  3. 届いた確認メールのリンクをクリックしてメール認証を完了させる
  4. パスワードを設定する（英数字・記号を含む8文字以上を推奨）
  5. 名前・会社名・業種・役職などのプロフィール情報を入力する
  6. アカウント作成完了。そのままチャットルームに案内される

プロフィールの設定

アカウント作成後に必ず行いたい初期設定です。プロフィールが整っていると、コンタクト申請の承認率も上がります。

設定項目設定場所ポイント
プロフィール画像右上アイコン → プロフィール顔写真または会社ロゴを設定しておくと信頼感が増す
表示名プロフィール設定フルネームを推奨。ニックネームだと社外からわかりにくい
所属・役職プロフィール設定社外との連絡では必須。「（株）〇〇 営業部 田中」のように記載
メールアドレスアカウント設定ログインメール。変更時はセキュリティ確認あり
言語設定アカウント設定日本語に設定されていることを確認（デフォルトで日本語）

アプリのインストール（推奨）

対応デバイスと推奨アプリ

ブラウザからも使用できますが、デスクトップアプリのインストールを推奨します。PCはWindows/Mac両対応。スマートフォンはiOS/Android対応のアプリが公式サイトからダウンロードできます。

コンタクト追加と管理

Chatworkは社内メンバーと社外コンタクトを同じ画面で管理できます。取引先・クライアントとのやり取りもChatwork1本に集約できる点が大きな強みです。

コンタクト追加の方法

方法手順向いているシーン
メールアドレスで招待コンタクト追加 → メールアドレスを入力 → 招待メール送信まだChatworkを使っていない相手に招待したい場合
Chatwork IDで検索コンタクト追加 → Chatwork IDを入力 → 申請送信相手が既にChatworkを使っている場合
QRコード自分のプロフィール → QRコードを表示 → 相手にスキャンしてもらう名刺交換・対面でのコンタクト追加に便利
グループチャット経由既存のグループチャットのメンバーをクリック → コンタクト申請共通のチャットルームにいる相手を追加したい場合

コンタクトの種類と権限

コンタクト種別説明
組織内メンバー同じ企業アカウントに登録されているメンバー。管理者が一括管理できる
組織外コンタクトコンタクト申請で承認した社外の取引先・クライアントなど
未承認のコンタクト申請申請を受けているがまだ承認していない相手。承認するまで会話は始まらない
コンタクト申請の注意点

見知らぬ相手からのコンタクト申請は、なりすまし詐欺の入り口になるケースがあります。申請者のプロフィール・所属会社を必ず確認してから承認してください。詳しくはセクション12「なりすまし詐欺対策」をご参照ください。

グループチャットの使い方

Chatworkの中心機能がグループチャットです。部署・プロジェクト・取引先など目的ごとにチャットルームを作成して会話を整理します。

グループチャットの作成

  1. 左側チャット一覧の上部にある「+」アイコンをクリックする
  2. 「グループチャットを作成」を選択する
  3. チャット名・アイコン・説明を設定する（後から変更可能）
  4. 招待するメンバーをコンタクトリストから選択する
  5. 「作成」をクリックして完了。即座にメッセージを送り始められる

チャット名の付け方（運用ポイント）

【チャット名の付け方の例】
「営業部_全体」            → 部署の常設チャット
「★ProjectA_開発チーム」  → ★をつけて重要チャットを上位表示
「【2026年3月】キャンペーン企画」→ 期間限定プロジェクト用
「山田さん_株式会社〇〇」  → 取引先との1:1

【目的・期間が一目でわかる名前にするのがコツ】
- 部署名を先頭に入れると一覧で見やすくなる
- 【】や★で重要度・緊急度をわかりやすく示す
- プロジェクト完了後はアーカイブで整理する

グループチャットの主な設定

設定項目説明設定場所
ピン留めよく使うチャットを一覧の上部に固定表示するチャット名を右クリック → ピン留め
非表示一時的にチャット一覧から非表示にする（退出はしない）チャット名を右クリック → 非表示
通知設定このチャットだけ通知をオフにするチャット右上の設定アイコン
メンバー管理新しいメンバーの追加・退出処理チャット上部の「メンバー」タブ
管理者権限特定メンバーに管理者権限を付与する（ビジネスプラン以上）メンバー設定 → 権限変更

09

ファイル共有・管理

Chatworkでは画像・PDF・Excelなど各種ファイルをチャット内に添付して共有できます。共有したファイルは「ファイル」タブからいつでも検索・再ダウンロードが可能です。

ファイルの送り方

  1. チャットのメッセージ入力欄の「クリップ」アイコンをクリックする
  2. 送りたいファイルを選択する（ドラッグ&ドロップでも可）
  3. コメントを追加したい場合はテキストを入力してから送信する

ファイルストレージの容量

プラン1ファイルの上限合計ストレージ
フリー5MB5GB（全体）
ビジネス10MB10GB/ユーザー
エンタープライズ10MB30GB/ユーザー
ファイル共有の注意点
  • 個人情報・機密情報を含むファイルの送付先は送信前に必ず確認する
  • フリープランは1ファイル5MBが上限。大容量ファイルはGoogleドライブ等のリンク共有を検討する
  • Chatwork上のファイルは削除すると相手側でもダウンロードできなくなる

ファイルの検索・再ダウンロード

チャットルームの「ファイル」タブを開くと、そのチャットで共有されたすべてのファイルが一覧表示されます。ファイル名で検索することも可能です。ビジネスプラン以上はメッセージ履歴が無制限なので、過去のファイルを永続的に管理できます。

通知設定

Chatworkは多くのチャットに参加していると通知が多すぎると感じるケースがあります。通知を適切に設定することで、業務に集中できる環境を作れます。

通知の種類

通知の種類説明
デスクトップ通知PCの画面右下（またはMac右上）にポップアップで表示される
サウンド通知メッセージ受信時に音が鳴る
メール通知未読メッセージがある場合に登録メールアドレスに通知が届く（頻度設定可）
バッジ（未読数）チャット一覧に未読件数の数字バッジが表示される

通知を絞る方法

【通知設定の推奨パターン】

■ パターン1：Toがついたメッセージのみ通知
設定場所：アカウント設定 → 通知 → 「Toがついたメッセージのみ通知する」をON
→ 自分宛てのメッセージだけ通知。情報過多を防ぐ基本設定

■ パターン2：特定チャットだけミュート
設定場所：対象チャットを右クリック → 「通知をオフにする」
→ 重要度の低いチャットだけを個別にミュート

■ パターン3：作業時間中だけDND（取り込み中）
設定場所：左下の自分のアイコン → 「取り込み中」に設定
→ 集中したい時間帯は「取り込み中」に切り替えると通知が抑制される

メール通知の設定

初期設定ではメール通知が有効になっていることがあります。Chatworkをメインで使っている場合はメール通知をオフにするとメールボックスがすっきりします。アカウント設定 → 通知設定 → メール通知の頻度を「受け取らない」に変更してください。

ビデオ・音声通話

Chatworkにはビデオ・音声通話機能が内蔵されており、チャットを開いたままオンライン会議を始められます。外部ツールを別途開く必要がありません。

通話の開始方法

  1. 通話したいチャットルームを開く
  2. 右上のビデオカメラアイコンをクリックする
  3. 「ビデオ通話を開始」または「音声通話を開始」を選択する
  4. 参加者に通話への参加リンクが通知される（自動通知）
  5. 相手が参加したら通話開始

通話中に使える機能

機能操作方法備考
マイクのミュート通話画面のマイクアイコンをクリック発言しないときはミュートを活用
カメラのオン/オフカメラアイコンをクリック音声のみにすることも可能
画面共有画面共有アイコンをクリック → 共有する画面を選択資料の共有・デモに便利
背景設定設定アイコン → 背景を選択バーチャル背景・ぼかしが利用可能
チャットテキスト通話中もチャット入力が可能URL・URL・リンクの共有に活用

通話に関する制限

項目フリービジネス/エンタープライズ
参加人数1対1のみ複数人対応（上限あり）
通話時間制限なし制限なし
画面共有利用可能利用可能
録画非対応非対応（外部ツールを使用）
通話品質を安定させるコツ

有線LAN接続または5GHz帯のWi-Fiを使用することで通話品質が向上します。カメラを使う場合は照明を顔の前方に配置すると映りが改善されます。

なりすまし詐欺対策

Chatworkを悪用したなりすまし詐欺が急増しています

2024〜2025年頃から、経営者・上司・取引先を装った偽アカウントでChatworkに接触し、急ぎの振込・情報提供などを要求する詐欺被害が多数報告されています。被害を未然に防ぐために、このセクションの内容を必ずチームに共有してください。

典型的な詐欺の手口

手口具体的な内容
社長・役員なりすまし「社長です。急ぎで〇〇万円を振り込んでください。詳細は後で説明します」
上司なりすまし「田中部長です。取引先への支払いを先に立替えてもらえますか」
取引先なりすまし「〇〇株式会社の担当者が変わりました。新しい口座番号をお伝えします」
IT部門なりすまし「システム部門です。セキュリティ確認のためにパスワードを教えてください」

詐欺かどうかを見分けるチェックポイント

  • 公式の社内連絡ルートでも同じ指示が来ているか確認した
  • 相手のChatwork IDが社内登録済みのIDと一致している
  • 電話・対面など別チャンネルで本人確認が取れている
  • 急かされる・他言無用などの圧力がない
  • 見知らぬアカウントから突然コンタクト申請が来た
  • 「急いで」「今すぐ」「内密に」など焦らせる言葉が使われている
  • 振込・送金・口座番号変更などの金銭絡みの要求がある
  • アカウント名は知っている人物だがIDが違う・プロフィール画像がない

被害を防ぐための社内ルール

【Chatwork詐欺対策 社内ルール（例）】

1. 金銭に関わる指示はChatworkのみで完結させない
   → 必ず電話・対面などの別手段で本人確認を行う

2. 急ぎの振込・口座変更依頼は即座に応じない
   → 上長への報告・確認を必須とする

3. コンタクト申請は既知の相手からのみ承認する
   → 見知らぬ相手からの申請はすぐに承認しない

4. 社内アカウントのIDリストを管理する
   → 正規アカウントのID一覧を共有フォルダなどで管理する

5. 怪しいと思ったら迷わず報告する
   → 詐欺と断定できなくても「怪しいアカウントがあった」と
     総務・情報システム部門に共有する

なりすましに遭った場合の対応

  1. 当該アカウントとのやり取りを即座に停止する（送金・情報提供を絶対にしない）
  2. 画面をスクリーンショットで保存する（証拠を残す）
  3. 社内の情報システム部門・総務部門に報告する
  4. 振込が完了してしまった場合は速やかに金融機関に連絡し振込停止を依頼する
  5. Chatwork上で当該アカウントをブロック・通報する（対象アカウントのプロフィール → 通報）
Chatwork公式サポートへの通報方法

なりすましアカウントはChatworkのサポート窓口に通報することができます。通報するには対象アカウントのプロフィール画面を開き「このユーザーを報告する」をクリックしてください。Chatwork社がアカウントの調査・停止措置を行います。

外部連携・API活用

Chatworkは他のサービスと連携することで、通知の自動化・業務フローの効率化が可能です。ノーコードでもすぐに始められる連携方法を中心に紹介します。

主な外部連携ツール

連携ツールできること設定の難易度
ZapierGoogleフォーム・Salesforce・Gmailなど900以上のサービスとChatworkを自動連携中（ノーコード）
Make（旧Integromat）複雑な条件分岐を含む自動化シナリオが作れる。Zapierより柔軟性が高い中〜高
Chatwork APIプログラムからメッセージ送信・ルーム作成などをAPI経由で操作できる高（開発者向け）

Chatwork APIの基本的な使い方

APIキーを発行することで、外部システムからChatworkにメッセージを自動送信できます。たとえば「フォームの送信時に特定のチャットルームに通知を送る」「夜間バッチ処理の完了をチャットに通知する」といった活用が可能です。

【Chatwork APIでメッセージを送信する例（curl）】

curl -X POST \
  -H "X-ChatWorkToken: あなたのAPIキー" \
  -d "body=テスト送信です" \
  "https://api.chatwork.com/v2/rooms/ルームID/messages"

【ルームIDの確認方法】
Chatworkの対象チャットを開いたときのURL末尾の数字がルームID
例：https://www.chatwork.com/#!rid123456 の場合、ルームIDは 123456
APIキーの取り扱い注意

APIキーが漏洩すると第三者が不正にメッセージを送信したりチャット内容を取得したりできます。GitHubなどの公開リポジトリには絶対にAPIキーをアップロードしないでください。環境変数で管理するのが基本です。

よくある質問とトラブル対応

アカウント・ログイン関連

質問・症状回答・対処法
パスワードを忘れたログイン画面の「パスワードを忘れた方はこちら」からメールで再設定する
メールアドレスを変更したいアカウント設定 → メールアドレス変更 → 旧メールアドレスと新メールアドレス両方に確認メールが届く
複数のデバイスで使いたい同じアカウントでPC・スマートフォンの両方にログインできる（同時ログイン可）
アカウントを削除したいアカウント設定 → アカウントの削除。削除するとすべてのデータが失われる（取り消し不可）

チャット・メッセージ関連

質問・症状回答・対処法
過去のメッセージが見えなくなったフリープランは30日分のみ表示。ビジネスプランにアップグレードすると全期間表示される
チャットルームを間違えて退出した管理者に再招待を依頼する。自分では戻れない
メッセージを誤送信した送信直後なら「...」→「削除」で取り消せる。相手が既読の場合でも削除は可能
特定のキーワードでメッセージを検索したい画面上部の検索ボックスにキーワードを入力して検索できる（ビジネスプランは全期間・フリーは30日分）

通知・表示関連

質問・症状回答・対処法
通知音が鳴らないアカウント設定 → 通知 → サウンドがONになっているか確認。PCのシステム音量も確認する
デスクトップ通知が表示されないブラウザの通知許可がONになっているか確認。Chromeの場合はアドレスバー左の錠前アイコンから確認できる
未読のバッジが消えない該当チャットを開いてすべてのメッセージを確認する。それでも消えない場合はアプリを再起動する
チャット一覧に表示されないチャットがある「非表示」に設定している可能性がある。チャット一覧の絞り込みを「全て」にして確認する

スマートフォンアプリ関連

質問・症状回答・対処法
スマートフォンに通知が届かないスマートフォンのアプリ設定で「通知を許可」がONになっているか確認する。iOSは設定アプリ → Chatwork → 通知
アプリが重い・動作が遅いアプリを一度終了して再起動する。改善しない場合はアプリをアップデートする
PCとスマートフォンでメッセージの既読がズレるどちらか一方で読んでも既読になるが、既読情報の同期にタイムラグが生じる場合がある。しばらく待つと解消されることが多い
公式サポートへの問い合わせ

本ガイドで解決できない問題はChatwork公式のヘルプセンター（support.chatwork.com）に詳細な情報があります。有料プランではチャットサポートも利用できます。
