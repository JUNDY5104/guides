01

Notionとは何か

Notionは、メモ・ドキュメント・タスク管理・データベース・Wikiなど、これまで複数のアプリに分散していた情報をひとつのワークスペースに集約できるオールインワン型の生産性ツールです。個人の日記から大企業の社内Wikiまで、幅広い用途で世界数千万人以上が活用しています。

2025年秋にはNotion 3.0が発表され、Notion AIが単なる文章アシスタントからAIエージェントへと進化しました。複雑なタスクを最大20分間、自律的に実行できるようになり、ツールとしての可能性が大きく広がっています。

Notionでできること

  • テキストメモ・日記・議事録・レポートの作成
  • データベースを使ったタスク管理・プロジェクト進捗管理
  • 社内Wiki・ナレッジベース・マニュアルの構築
  • チームでのリアルタイム共同編集・ページ共有
  • フォーム作成・アンケート収集（Notionフォーム）
  • Webサイト・ポートフォリオの公開（Notionサイト）
  • AI機能による文書作成・要約・議事録の自動生成
  • 他ツール（Slack・Google Drive・GitHubなど）との連携

アカウント作成と初期設定

Notionはブラウザ・デスクトップアプリ・スマートフォンアプリのいずれからでも始められます。無料プランから始めて、必要に応じて有料プランへ移行できます。

N
Notionへようこそ
メールアドレスを入力
メールアドレスで続ける
または
G    Googleで続ける
A    Appleで続ける
続けることで、利用規約とプライバシーポリシーに同意したことになります

アカウント作成の手順

  1. notion.com にアクセスし、「Notionを無料で使う」をクリックする
  2. メールアドレス・Googleアカウント・Appleアカウントのいずれかで登録する
  3. メール登録の場合は届いた確認メールのリンクをクリックして認証する
  4. 利用目的（個人・チームなど）を選択し、ワークスペース名を設定する
  5. ホーム画面が表示されれば登録完了。フリープランで即利用可能

日本語表示への切り替え

  1. 左サイドバー下部のアカウント名をクリックし、「設定」を選択する
  2. 「Language & region（言語と地域）」を開く
  3. 「日本語」を選択してページをリロードする

アプリの種類と使い分け

種類特徴おすすめ場面
ブラウザ版インストール不要。どのPCでも使える初めて試すとき・出先での作業
デスクトップアプリ（Mac/Win）動作が安定。オフライン対応（有料プランで拡張）メインPCでの日常作業
モバイルアプリ（iOS/Android）外出先・スキマ時間での確認・メモスマートフォンでのクイックメモ
2025年新機能：パスキー認証 2025年から指紋認証・Touch ID・Windows Helloなどのパスキーでのログインに対応。パスワード不要で素早く安全にサインインできるようになりました。

ページとブロックの基本

Notionのコンテンツは「ページ」と「ブロック」の2層構造です。すべてのコンテンツ（テキスト・画像・リスト・表など）は「ブロック」という単位で管理されます。

ページの基本操作

  1. 左サイドバーの「+ 新規ページ」をクリックして新しいページを作成する
  2. タイトルを入力する（タイトル行はEnterで本文エリアへ移動）
  3. 本文エリアに「/」を入力して挿入したいブロックタイプを選ぶ
  4. ページ内にサブページを作ると、情報を階層的に整理できる（推奨：3階層以内）

主要ブロックの種類と使い方

コマンドブロック名使い方・用途
/textテキスト通常の本文。最もよく使う基本ブロック
/h1〜/h3見出し1〜3セクションの見出し。階層構造に応じて使い分ける
/todoToDoリストチェックボックス付きタスクリスト
/bullet箇条書きリスト点（丸）付きのリスト
/numbered番号付きリスト手順・ステップの説明に最適
/toggleトグルリストクリックで展開・折りたたみできる。FAQや詳細説明に便利
/tableシンプルテーブルデータベース機能なしの基本的な表
/databaseデータベース高機能なデータ管理。後述のSection 06で詳説
/image画像ファイルアップロードまたはURLで画像を挿入
/video動画YouTubeやVimeoのURLを貼り付けて埋め込み
/codeコードブロックプログラムコードのシンタックスハイライト表示
/quote引用引用文・メモを視覚的に目立たせる
/calloutコールアウト重要な注意書きやポイントを強調表示
/divider区切り線セクション間の水平線
/pageサブページ現在のページ内にネストしたページを作成
/aiAIブロックAI機能を呼び出してコンテンツを生成（後述）

ブロックの操作方法

  • 移動：ブロック左に表示される「::（6点ドット）」をドラッグして自由に並べ替え
  • メニュー：同アイコンをクリックすると「削除・複製・色変更・ブロックタイプ変更」などのメニューが表示
  • 複数選択：Shift+クリック、またはShift+ドラッグで複数のブロックを同時選択して一括操作
  • インラインコメント：テキストを選択して「コメントを追加」でそのテキストにコメントを付ける
2025年更新情報 AIブロック（/ai）は2025年9月のアップデートでプロンプトごと保存されるようになりました。一度作成したAIブロックは毎回入力し直す必要がなく、ページ内容の変化に合わせて自動的に再生成されます。

スラッシュコマンド（/コマンド）の活用

本文内で「/」を入力するとブロック挿入メニューが表示されます。キーワードで絞り込めるため、「/cal」でカレンダー、「/gall」でギャラリービューのデータベースなど、途中まで入力してすばやく目的のブロックを挿入できます。

@メンション・[[リンク]]の活用

@ページ名    → ページへのリンクを挿入（ページメンション）
@人名        → チームメンバーをメンション（通知が届く）
@日付        → 日付を挿入（リマインダー設定も可能）
[[ページ名]] → 既存ページへのリンクを挿入
+ページ名    → 新規ページへのリンクを作成
時短テクニック テキストを入力してからドラッグ選択し、「/」を押すとそのテキストを別のブロックタイプに変換できます。また選択状態でURLをペーストすると自動的にリンクに変換されます。

データベースの作成方法

  1. ページ本文で「/database」と入力する
  2. 「インライン」（ページ内に埋め込み）または「フルページ」（単独ページ）を選択する
  3. 列（プロパティ）を設定し、行（レコード）にデータを入力する
  4. 必要に応じてビューを追加し（次セクション参照）、用途に合った表示に切り替える

08

データベースの応用機能

データベースはビューの切り替えだけでなく、フィルター・ソート・グルーピング・リレーションなど高度な機能を組み合わせることで、業務レベルの情報管理が実現します。

フィルター機能

特定の条件に合ったデータだけを表示します。ビュー右上の「フィルター」から設定できます。複数条件の AND / OR 組み合わせも可能です。

【フィルター設定例】
条件1：ステータス が「進行中」
条件2：担当者 が「自分」
条件3：期限 が「今週以内」

→ 上記3条件をANDで組み合わせると
  「自分が担当し今週中に締め切りのある進行中タスク」だけが表示される

ソート機能

指定したプロパティの値で並べ替えます。「期限が近い順」「優先度が高い順」などの並べ替えが可能で、複数のソート条件を重ねることもできます。

グルーピング

セレクト・マルチセレクト・人物プロパティなどの値でレコードをグループ分けして表示します。ボードビューのステータス別表示はグルーピングの一種です。

リレーション（関連付け）

2つのデータベース間を関連付けます。例えば「プロジェクト」データベースと「タスク」データベースをリレーションで結ぶと、プロジェクトのページからそのタスクが、タスクのページからプロジェクト情報が参照できます。

【リレーション活用例】
プロジェクトDB ← リレーション → タスクDB
                                   ↓
                              ロールアップで集計
                  「完了タスク数 / 総タスク数」を自動表示

数式プロパティ

他のプロパティの値を参照して計算する高度な機能です。

【数式の使用例】

// 期限まであと何日か表示
dateBetween(prop("期限"), now(), "days")

// 完了チェックに応じたステータスラベル
if(prop("完了"), "Done", "In Progress")

// 売上達成率を計算
round(prop("実績") / prop("目標") * 100)

オートメーション（自動化）

データベース内のプロパティ変化をトリガーにして、特定のアクションを自動実行する機能です（プラスプラン以上）。

【オートメーションの例】
トリガー：ステータスが「完了」に変更された
アクション：「完了日時」プロパティに現在時刻を自動入力

トリガー：新しいページが作成された
アクション：担当者プロパティに「自分」を自動アサイン
注意：オートメーションはプラスプラン以上 データベースのオートメーション機能（自動化）はプラスプラン以上で利用できます。フリープランでは手動で値を変更する必要があります。

テンプレートの活用

ゼロから設計する手間を省くために、Notionには公式テンプレートと自作テンプレートの2種類が用意されています。

公式テンプレートの使い方

  1. 左サイドバーの「テンプレート」をクリックする（または本文で「/template」と入力）
  2. カテゴリ（仕事・学習・個人など）から目的のテンプレートを探す
  3. 「このテンプレートを使用」をクリックすると複製されてすぐ使える
  4. 内容を自分の用途に合わせてカスタマイズする

おすすめ公式テンプレート

テンプレート名カテゴリ内容・活用シーン
タスクリスト仕事シンプルなToDoとプロジェクト管理
週次レビュー個人週ごとの振り返りと目標設定
読書リスト学習読んだ本・読みたい本の記録
会議メモ仕事アジェンダ・議事録・アクションアイテムの管理
習慣トラッカー個人毎日の習慣の達成状況を記録
目標管理（OKR）仕事四半期・年間目標と達成率の管理
クライアント管理仕事顧客情報・案件進捗の一元管理

自作テンプレートの作り方

データベース内に「テンプレート」を登録すると、新規ページ作成時に同じ構造のページが自動挿入されます。

  1. データベースの右上「新規」ボタン横の「v（矢印）」をクリックする
  2. 「テンプレートを作成」を選択する
  3. テンプレートとして使いたい内容（見出し・チェックリスト・プロパティ初期値など）を入力して保存
  4. 次回「新規」を押すとテンプレートが反映された状態でページが作成される

FAQ・トラブルシューティング

Notionを使い始めたときによく出る疑問・トラブルをまとめました。

よくある質問

誤って削除したページを復元できますか？
はい。削除したページは「ゴミ箱」に保存されます。左サイドバー下部の「ゴミ箱」をクリックし、対象ページを選んで「復元」をクリックしてください。フリープランは7日間、プラスは30日間、ビジネスは90日間、エンタープライズは無制限で復元できます。
オフラインでも使えますか？
デスクトップアプリ・モバイルアプリでは2025年に正式実装されたオフラインモードが利用できます。有料プランではお気に入りや最近アクセスしたページが自動的にオフライン対応になります。オフライン中の変更はオンライン復帰後に自動同期されます。
他のツールからデータをインポートできますか？
はい。Evernote・Confluence・Trello・CSV・HTMLなどからインポートできます。設定メニューの「インポート」から対応フォーマットを確認してください。なお、インポートの完了には大量データの場合時間がかかることがあります。
データはバックアップできますか？
設定メニューの「エクスポート」からワークスペース全体をHTML・Markdown・CSV形式でバックアップできます。ワークスペース全体のエクスポートはビジネスプラン以上の機能です。フリー・プラスはページ単位でのエクスポートが可能です。
スマートフォンでも使いやすいですか？
iOS・Android向けの専用アプリがあり、基本操作は問題なく行えます。ただし複雑なデータベース操作や大量の編集作業はPCのほうが効率的です。モバイルはクイックメモ・確認・コメント追加などの用途に適しています。
フリープランで複数人で使うと制限がありますか？
はい。フリープランをチームで使う場合、ブロック数の制限（上限1,000ブロック）があります。個人利用ではこの制限はありません。チームで本格的に使いたい場合はプラスプランへのアップグレードをおすすめします。
Notion AIは日本語に対応していますか？
はい。Notion AIは日本語を含む多言語に対応しています。日本語での文章生成・要約・翻訳・質問応答が可能です。Notion 3.0のAIエージェント機能も日本語で操作できます。
Notionのデータは安全ですか？
NotionはSOC2 Type II認証を取得しており、保存・転送中のデータは暗号化されています。また、GDPRおよびCCPAに準拠しています。エンタープライズプランでは監査ログ・IP制限・SAML SSOなど追加のセキュリティ機能が利用できます。
Notionメールとは何ですか？
GmailやGoogle Workspaceのメールを管理できるNotion内のメーラー機能です。Notionのビュー機能を使って「優先度が高いメール」「採用関連メール」など目的別の受信トレイを作れます。Notionカレンダーとの連携による日程調整も可能です。最新のプラン情報については公式サイトをご確認ください。
ページが重い・動作が遅いときはどうすればよいですか？
1ページにコンテンツを詰め込みすぎるとパフォーマンスが低下します。対策として「トグルリストに内容を折りたたむ」「サブページに分割する」「埋め込みコンテンツを減らす」ことが有効です。デスクトップアプリのほうがブラウザ版より安定した動作になる場合があります。
公式サポートリソース Notionの公式ヘルプセンター（notion.com/ja/help）では詳細なドキュメントが日本語で提供されています。機能の最新情報はリリースノート（notion.com/ja/releases）で確認できます。
