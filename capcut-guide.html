CapCutとは・他ツールとの比較

CapCut（キャップカット）はTikTokを運営するByteDance社が提供する動画編集ツールです。2019年のリリース以降、スマートフォンアプリとして爆発的に普及し、現在はPC版・Web版も提供されています。

「動画編集を始めるならまずCapCutから」と言われるほど初心者向けに設計されており、無料でも基本的な編集機能をほぼすべて使えます。TikTok・Instagram Reels・YouTube Shortsなどのショート動画制作に特に強みを持っています。

CapCutが特に向いている用途

  • TikTok・Instagram Reels・YouTube Shorts： 縦型ショート動画のテンプレートが豊富でトレンドに即対応
  • AI字幕・テロップ自動生成： 音声を自動で文字起こしして字幕を一括追加
  • 動画編集初心者： 複雑な設定なしに直感的な操作ですぐ始められる

インストール・初期設定

CapCutはPC版・スマホ版・Web版の3種類があります。それぞれ特徴が異なるため、用途に合わせて選んでください。

PC版（デスクトップ）のインストール

  1. capcut.com にアクセスして「ダウンロード」をクリックする
  2. Windows版またはMac版を選択してインストーラーをダウンロードする（Microsoft Storeからも入手可）
  3. インストーラーを実行してインストールを完了させる
  4. Google・TikTok・メールアドレスのいずれかでログインする
  5. 初回起動後、言語設定で日本語が選択されていない場合は「Settings」→「General」→「日本語」→「Save」で変更する

スマホ版のインストール

  1. App Store（iPhone）またはGoogle Play（Android）で「CapCut」を検索する
  2. ByteDance Ltd.の公式アプリをインストールする（類似アプリに注意）
  3. アプリを起動してGoogleアカウントまたはTikTokアカウントでログインする
PC版とスマホ版でアカウントを共通化 同じアカウントでログインすることで、プロジェクトや素材をクラウドで同期できます。スマホで撮影して、PCで本格編集するというワークフローが可能です。ただしPro版をモバイルから購入するとモバイルのみの適用になるため、PC・スマホ両方で使いたい場合はデスクトップ版から購入してください。

基本編集の操作方法

CapCutの基本編集はタイムライン上でクリップを操作するだけです。複雑なコマンドは不要で、ドラッグ操作でほとんどの編集が完結します。

新規プロジェクトの作成から編集まで

  1. ホーム画面の「+新しいプロジェクト」をクリックして動画ファイルを読み込む
  2. 読み込んだ素材をタイムラインにドラッグ＆ドロップで配置する
  3. タイムライン上のクリップの端をドラッグしてトリミング（カット）する
  4. クリップを選択して右クリック→「分割」または「Ctrl+B」でカット編集する
  5. クリップをドラッグして順番を入れ替える
  6. 編集が完成したら右上の「エクスポート」で書き出す

よく使う編集操作のショートカット

【PC版 必須ショートカット】
Ctrl + Z         : 元に戻す
Ctrl + Shift + Z : やり直す
Ctrl + B         : 再生ヘッドの位置でクリップを分割（カット）
Ctrl + C / V     : コピー・貼り付け
Delete           : 選択中のクリップを削除
Space            : 再生・一時停止
←→ 矢印キー      : フレーム単位で移動
Shift + ←→      : 1秒単位で移動
Ctrl + +/-       : タイムラインのズームイン・アウト
Ctrl + E         : エクスポート画面を開く

【Mac版は Ctrl → Command に読み替え】

速度変更・スロー・タイムラプスの作り方

【速度変更の手順】
1. タイムライン上のクリップを選択する
2. 右パネルの「速度」タブをクリック
3. 「通常」または「曲線」を選択する
   - 通常：全体を一定の速度で変更（0.1x〜100x）
   - 曲線：シーン内で速度を変化させる（ドラマチックなスロー表現に）

スロー（0.5x）   : ドラマチックな演出・感情的なシーン
タイムラプス（4x+）: 景色・料理・作業の過程を短縮
ハイスピード（2x）: テンポ感を上げたい時・冗長なシーンの短縮

テキスト・テロップの追加

CapCutのテキスト機能は充実しており、AI自動字幕から手動テロップまで様々な方法で文字を追加できます。SNS動画では字幕の有無が視聴完了率に大きく影響します。

テキスト追加の3つの方法

方法内容おすすめ場面
手動テキスト左パネル「テキスト」→「テキストを追加」。好きな位置・フォント・色で配置タイトル・強調テキスト
AI自動字幕（Pro推奨）音声を自動認識して字幕を生成。日本語対応・一括編集可能トーク動画・インタビュー・Vlog
テキストテンプレートデザイン済みのアニメーション付きテキストをワンクリックで適用SNS向けスタイリッシュなテロップ

AI自動字幕の使い方

  1. 左パネルの「テキスト」→「自動キャプション」をクリックする
  2. 言語「日本語」を選択して「キャプションを生成」をクリックする
  3. 音声を自動で認識して字幕が生成される（1〜3分程度）
  4. 生成された字幕をクリックして誤りを修正する
  5. 全字幕を選択して一括でフォント・色・サイズを変更できる
テキストアニメーションのコツ テキストを選択して右パネルの「アニメーション」タブから入場・ループ・退場のアニメーションを設定できます。SNS動画ではアニメーション付きテキストの方が視聴者の目を引きやすいです。持続時間は0.3〜0.5秒程度のテンポの良い設定がおすすめです。

音楽・音声・効果音の使い方

CapCutには著作権フリーの音楽ライブラリが内蔵されており、TikTokのトレンド楽曲にもアクセスできます。音声編集はSNS動画の完成度を大きく左右します。

BGMの追加方法

  1. 左パネルの「音楽」をクリックして内蔵ライブラリを開く
  2. ジャンル・ムード・テンポで絞り込んで楽曲を選ぶ
  3. 使用したい楽曲の「+」ボタンをクリックしてタイムラインに追加する
  4. 音声クリップの端をドラッグして長さを調整する
  5. 右パネルで音量・フェードイン・フェードアウトを設定する

音声編集の主要機能

AI機能完全解説

CapCutのAI機能は2026年時点で最も充実した動画編集ツールの一つです。かつては数時間かかっていた作業がワンクリックで完了します。

AI自動カット Pro推奨

無音部分・シーンの切り替わり・不要な間を自動検出してカット。ゲーム実況・配信アーカイブの編集時間を大幅短縮。PC版は「ツール」→「スマートカット」からアクセス

背景削除 Pro

グリーンバックなしに人物・商品の背景をAIで自動削除。透明背景のクリップを生成して別の背景と合成できる。精度は高く、髪の毛の細部まで対応

自動字幕 無料

音声を自動認識して字幕を生成。日本語・英語など多言語対応。生成後に一括でフォント・色・サイズを変更可能。手動修正も簡単

ボイスクローン Pro

自分の声を学習してAIが再現。テキストを入力するとクローン音声で読み上げ。ナレーション収録の手間を大幅に削減

AI画像生成 Pro

テキストプロンプトからAIが画像を生成。サムネイル・背景素材・スライド画像の制作に活用できる

ボーカル分離 Pro

楽曲から歌声だけ・伴奏だけを分離するAI機能。カラオケ音源の作成やBGMの声被り解消に使える

スマートリフレーム 無料

横長動画を縦型（9:16）に自動変換。被写体を追跡してフレーミングを自動調整するため、手動でのトリミングが不要

フリッカー除去 Pro

蛍光灯・モニター撮影時に発生するフリッカー（ちらつき）をAIが自動除去。室内撮影の動画品質を向上

AIクレジット（月800クレジット）について Pro版では月800クレジットが付与され、AI画像生成・ボイスクローンなどクレジット消費機能に使用します。自動字幕・背景削除・AI自動カットはクレジット不要で使い放題です。

エフェクト・トランジション・フィルター

CapCutは無料版でも数百種類のエフェクト・トランジションが使えます。Pro版ではさらに600種類以上のエフェクトに加え、TikTokトレンドと連動した最新テンプレートが随時追加されます。

エフェクトの使い方

  1. 左パネルの「エフェクト」をクリックしてライブラリを開く
  2. 「動画エフェクト」または「ボディエフェクト」から選択する
  3. 使用したいエフェクトを選んでタイムラインに追加する
  4. エフェクトの長さ・強度を右パネルで調整する

トランジション（シーン切り替え）の追加

【トランジションの追加手順】
1. タイムライン上の2つのクリップの「つなぎ目」にある
   白いアイコン（|）をクリックする
2. トランジションライブラリが開く
3. 使いたいトランジションを選択してプレビュー確認
4. 「時間」スライダーで切り替えの長さを調整（0.1〜2秒が目安）

【よく使うトランジションの選び方】
自然な切り替え  → フェード・クロスディゾルブ
テンポある演出 → スライド・ズーム
インパクト系   → グリッチ・フラッシュ
おしゃれ系     → スムーズ・モーションブラー（Pro）

【トランジションを全クリップに一括適用】
「すべてに適用」ボタンをクリックすると
タイムライン全体に同じトランジションを一括で設定できる

カラーフィルター・色補正

  • フィルター： ワンクリックで映画風・ヴィンテージ・クール・ウォームなど雰囲気を変えられる
  • 調整： 明るさ・コントラスト・彩度・シャープネスなどを数値で微調整
  • カラーマッチ（Pro）： 複数クリップのカラーを自動的に統一する機能

書き出し・SNS投稿

編集が完了したら書き出し（エクスポート）を行います。書き出し設定によって画質・ファイルサイズ・透かしの有無が変わります。

書き出し設定の選び方

設定項目推奨設定注意点
解像度1080p（Full HD）4KはPro版のみ。SNSは1080pで十分
フレームレート30fps（または24fps）60fpsはスポーツ・ゲーム動画向け
ビットレート推奨（バランス）高いほど画質は良いがファイルサイズが増える
形式MP4（H.264）最も互換性が高い。SNS投稿の標準形式
透かし削除（Pro版）無料版はCapCutロゴが左上に表示される

SNSへの直接投稿

書き出し後の画面またはエクスポート設定から、TikTok・Instagram・YouTubeなどに直接投稿できます。

  • TikTok： CapCutとTikTokは同じByteDance製のため連携がシームレス。キャプション・ハッシュタグもCapCut内で入力できる
  • Instagram： Reels・ストーリーズのサイズに合わせた書き出しが自動で選択される
  • YouTube： タイトル・説明文を入力してそのまま投稿できる
透かし（ウォーターマーク）について 無料版で書き出した動画には左上に「CapCut」のロゴが表示されます。SNSでの本格的な活用を考えているなら、透かし削除はPro版に課金する最大の理由のひとつです。7日間の無料トライアルで試すことができます。

よくあるトラブル・FAQ

書き出した動画にCapCutのロゴが入っています。消す方法はありますか？
無料版の仕様です。ロゴを消すにはPro版（月額1,380円）へのアップグレードが必要です。7日間の無料トライアルで試すことができます。購入はcapcut.comのウェブサイトからが最安です。
書き出し後の動画が編集中より画質が悪く見えます。
書き出し設定の解像度・ビットレートを上げることで改善できます。エクスポート設定で「1080p」「高品質」を選択してください。また、SNSにアップロードする際に再圧縮されるため、元ファイルより少し画質が落ちることは避けられません。無料版は最大1080pまでです。
AI自動字幕の精度が低く、誤字が多いです。
ノイズの少ない環境で録音した音声ほど精度が上がります。滑舌を意識してゆっくり話すと認識率が向上します。また、専門用語・固有名詞は自動認識が苦手なため、生成後に手動で修正してください。字幕テキストをクリックすると直接編集できます。
動作が重くて編集中にカクつきます。
プレビュー品質を下げることで解決できます。プレビュー画面右上の解像度設定を「1/2」または「1/4」に変更してください。書き出し時は元の品質で出力されます。また、不要なアプリを閉じてメモリを解放することも有効です。
商用利用はできますか？
自分で撮影した映像をCapCutで編集して商用利用することは基本的に可能です。ただし、CapCutの内蔵素材（音楽・エフェクト・テンプレート・ストック映像）には個別にライセンスが設定されており、商用利用が制限されているものもあります。重要な商業案件で使用する前に各素材のライセンス条件を公式で確認してください。
Pro版を解約する方法は？
解約は自動更新の24時間前までにアカウント設定から行う必要があります。ウェブ版で購入した場合はcapcut.comのアカウント設定→サブスクリプションから解約できます。App Store・Google Play経由で購入した場合は各ストアのサブスクリプション管理画面から解約してください。解約後も有効期限まではPro機能が使えます。
スマホで購入したPro版がPCで使えません。
モバイルアプリから購入したPro版はモバイルのみ有効です。PC版でも使いたい場合はcapcut.comのウェブサイトからデスクトップ版のProプランを別途購入する必要があります。PC版で購入するとスマホでも使えます。
