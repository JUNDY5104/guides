Notionとは

Notionは、メモ・タスク管理・データベース・Wiki・カレンダーなど、複数の機能を1つのツールにまとめたオールインワン型の生産性アプリです。個人利用から大企業のチームまで幅広く使われており、世界中で数千万人が活用しています。

Notionでできること

  • タスク管理・プロジェクト進捗の追跡
  • 表（データベース）による情報整理
  • チームでの共同編集・ページ共有
  • ポートフォリオや社内Wikiとしての活用

アカウント作成と初期設定

Notionはブラウザ・デスクトップアプリ・スマートフォンアプリのいずれからでも利用できます。まず無料アカウントを作成し、日本語表示に設定しましょう。

アカウント作成の手順

  1. notion.so にアクセスし、「Notionを無料で使う」をクリックする
  2. メールアドレス、またはGoogleアカウント・Appleアカウントで登録する
  3. 届いたメールのリンクをクリックして認証する（メール登録の場合）
  4. 利用目的（個人・チームなど）を選択し、ワークスペースを作成する
  5. ログイン完了。ホーム画面が表示される

日本語表示に変更する方法

Notionは初期設定が英語になっている場合があります。以下の手順で日本語に切り替えてください。

  1. 左サイドバー下部のアカウント名をクリックする
  2. 「Settings（設定）」を選択する
  3. 「Language & region」から「日本語」を選択する
  4. 画面を再読み込みすると日本語表示になる

画面の主な構成

エリア 内容
左サイドバー ページ一覧・検索・設定・ゴミ箱へのアクセス
メインエリア 選択中のページの編集・閲覧
右上ツールバー 共有・コメント・プロパティの表示切り替え

ページとブロックの基本

Notionのコンテンツはすべて「ページ」と「ブロック」で構成されています。ブロックとはテキスト・画像・リストなど、あらゆるコンテンツの最小単位です。

ページの作成方法

  1. 左サイドバーの「+ 新規ページ」をクリックする
  2. タイトルを入力する（Enterで確定）
  3. 本文エリアをクリックしてコンテンツを入力し始める
Tip ページの中にページを作ることができます。「/page」と入力するとサブページを挿入でき、階層的な情報整理に役立ちます。

ブロックの種類

本文エリアで「/」を入力すると、挿入できるブロックの一覧が表示されます。主なブロックは以下のとおりです。

コマンド ブロックの種類 用途
/text テキスト 通常の本文入力
/h1 見出し1 大見出し（最大フォントサイズ）
/todo ToDoリスト チェックボックス付きのタスクリスト
/bullet 箇条書きリスト 点（丸）付きのリスト
/table シンプルテーブル 行列形式の表（データベース機能なし）
/image 画像 ファイルアップロードまたはURL指定で画像を挿入
/code コードブロック プログラムコードのハイライト表示
/divider 区切り線 セクション間の水平線

ブロックの操作（コピー・移動・削除）

  • ブロック左の「6つの点（::）」アイコンをドラッグすると、ブロックを自由に並べ替えられます
  • 同じアイコンをクリックすると「削除・複製・色変更」などのメニューが表示されます
  • テキストを選択して「Ctrl+B」（Mac: Cmd+B）で太字にするなど、マークダウン的な書式設定が使えます

よく使うショートカット

Ctrl + N   : 新規ページを作成
Ctrl + P   : ページ検索（クイック検索）
Ctrl + /   : ブロックタイプを変更
Ctrl + D   : ブロックを複製
Ctrl + Z   : 元に戻す
[ ]        : ToDoチェックボックスを作成（半角角括弧2つ）
---        : 区切り線を挿入（ハイフン3つ）

※ Mac の場合は Ctrl を Cmd に読み替えてください

データベースの作成方法

  1. ページ本文で「/database」と入力する
  2. 「データベース - インライン」または「データベース - フルページ」を選択する
  3. 列名（プロパティ）を設定し、行にデータを入力する

ビューの種類と使い分け

ビュー 表示形式 向いている用途
テーブル 行・列の表形式 データの一覧管理・一括編集
カンバン カード形式（ステータス別） タスクの進捗管理（未着手・進行中・完了など）
カレンダー 月カレンダー形式 日付プロパティを持つ予定・締め切りの管理
ギャラリー カードのタイル表示 ポートフォリオ・レシピ・書籍リストなど
リスト シンプルな縦リスト メモや記事の一覧をすっきり表示

プロパティ（列）の種類

データベースの列にはさまざまな「型」を設定できます。

  • テキスト：自由記述のメモ欄
  • 数値：金額・数量などの数字データ
  • セレクト：あらかじめ決めた選択肢から1つ選ぶ（ステータス管理など）
  • マルチセレクト：複数のタグを付与できる
  • 日付：開始日・期限日などの日付指定
  • チェックボックス：完了フラグとして使う
  • リレーション：他のデータベースのレコードと関連付ける
活用例 タスク管理では「名前・担当者・ステータス（セレクト）・期限（日付）・優先度（セレクト）」のプロパティを組み合わせるのが基本構成として使いやすいです。

05

テンプレートの活用

Notionにはすぐに使える公式テンプレートが多数用意されており、ゼロから設計する手間を省けます。また、自分でよく使う構成をテンプレート化することも可能です。

公式テンプレートの使い方

  1. 左サイドバーの「テンプレート」をクリックする（またはページ本文内で「/template」と入力する）
  2. カテゴリ（仕事・学習・個人など）から目的のテンプレートを探す
  3. 「このテンプレートを使用」をクリックすると、ページとして複製される
  4. 複製されたページを自分の用途に合わせて編集する

おすすめ公式テンプレート

テンプレート名 用途
タスクリスト シンプルなToDoとプロジェクト管理
週次レビュー 週ごとの振り返りと目標設定
読書リスト 読んだ本・読みたい本の記録
会議メモ アジェンダ・議事録・アクションアイテムの管理
習慣トラッカー 毎日の習慣の達成状況を記録

自作テンプレートの作り方

データベース内に「テンプレート」を登録しておくと、同じ構造のページを毎回一から作らずに済みます。

  1. データベースの右上にある「新規」ボタン横の「v（矢印）」をクリックする
  2. 「テンプレートを作成」を選択する
  3. テンプレートとして使いたい内容（見出し・チェックリスト・プロパティなど）を入力して保存する
  4. 次回から「新規」をクリックするとテンプレートが反映された状態でページが作成される
自作テンプレートが便利なシーン
  • 毎週同じフォーマットで日報・週報を書くとき
  • プロジェクトごとに同じ管理シートを使うとき
  • 面談記録・議事録など定型フォーマットが決まっているとき

実践！活用シーン例

ここでは「タスク管理」「日記」「情報整理」という3つのシーンで、Notionをどのように構成すると使いやすいかを解説します。

シーン1：タスク管理

データベースを使ってプロジェクトのタスクを一元管理します。

【推奨プロパティ構成】
- 名前（タイトル）  : タスクの内容
- ステータス        : セレクト（未着手 / 進行中 / 完了）
- 担当者            : 人物（チーム利用時）
- 期限              : 日付
- 優先度            : セレクト（高 / 中 / 低）
- 関連プロジェクト  : セレクトまたはリレーション

【推奨ビュー構成】
- カンバンビュー（ステータス別）で進捗を視覚的に把握
- カレンダービュー（期限）で締め切りを確認
- テーブルビューで全タスクを一覧・フィルタ

シーン2：日記・ジャーナル

毎日の記録を継続するには、日付プロパティを持つデータベースが最適です。

【推奨プロパティ構成】
- 名前（タイトル）  : 日付（例：2025年6月1日）
- 日付              : 日付プロパティ（カレンダービュー用）
- 気分              : セレクト（良い / 普通 / イマイチ）
- 今日の出来事      : テキスト（本文）
- メモ              : テキスト（補足）

【テンプレート例（本文内容）】
## 今日の3行まとめ
1. 
2. 
3. 

## 感謝したこと

## 明日やること

シーン3：情報整理（ナレッジベース）

学んだことやリンク集をデータベースで整理し、後から検索・参照しやすくします。

【推奨プロパティ構成】
- 名前（タイトル）  : 情報・記事のタイトル
- カテゴリ          : マルチセレクト（AI / マーケ / 読書 など）
- URL               : URLプロパティ（外部リンク）
- 登録日            : 日付（自動入力）
- 重要度            : セレクト（高 / 中 / 低）
- メモ              : テキスト（要点のまとめ）

【活用のコツ】
- ギャラリービューでサムネイル付きのカード一覧にする
- カテゴリでフィルタをかけてジャンル別に絞り込む
Note Notionの検索機能（Ctrl+P / Cmd+P）は全ページ横断で機能するため、ページが増えても情報にすぐアクセスできます。キーワードを入れるだけで関連ページが一覧表示されます。

よくある質問と注意点

Notionを使い始めたときに多く寄せられる疑問と、無料プランの制限について整理します。

プランの違い（2025年現在）

プラン 対象 主な制限・特徴
フリー 個人 ページ数・ブロック数に制限なし。ゲスト招待は最大10名まで
プラス 個人・小規模チーム 無制限のゲスト招待・バージョン履歴（30日）
ビジネス チーム SAML SSO・高度な権限設定・バージョン履歴（90日）
エンタープライズ 大企業 無制限バージョン履歴・監査ログ・専用サポート
無料プランでできること 個人利用であれば無料プランで十分な機能が使えます。ページ数・データベース数の上限はなく、PC・スマートフォン・タブレットからのアクセスもすべて無料です。

よくある質問

オフラインでも使えますか？
デスクトップアプリ・モバイルアプリでは、一度開いたページはオフライン時でも閲覧・編集が可能です。ただし、変更はオンライン復帰後に同期されます。ブラウザ版はオフライン動作に制限があります。
他のアプリからデータをインポートできますか？
はい。Evernote・Notion独自形式（CSV・HTML）・Confluence・Trelloなどからインポートができます。設定メニューの「インポート」から対応フォーマットを確認してください。
ページを誤って削除してしまいました。復元できますか？
はい。削除したページは「ゴミ箱」に30日間保存されます。左サイドバー下部の「ゴミ箱」をクリックし、対象ページを選んで「復元」をクリックしてください。
ページを外部に公開することはできますか？
可能です。ページ右上の「共有」をクリックし、「Webに公開」をオンにするとURLが発行されます。アカウントを持っていない人でも閲覧できるようになります（公開範囲は設定で調整できます）。
スマートフォンから使いやすいですか？
iOS・Android向けのアプリが提供されており、基本的な操作はモバイルでも問題なく行えます。ただし、複雑なデータベース操作や大量のページ編集はPCのほうが作業しやすいです。
Notionと連携できるツールはありますか？
Slack・Google Drive・GitHub・Figmaなど多数のサービスと連携できます。また、Zapierを経由すると対応外サービスともある程度自動化が可能です。連携設定は各ページの「統合」メニューから行えます。
使いこなすための3つのコツ
  1. まず1つの用途（タスク管理か日記）だけで使い始める。最初から完璧な構成を目指さない
  2. ページの階層は3階層以内に収めると、情報が迷子になりにくい
  3. テンプレートを積極的に活用し、定型作業の入力コストを下げる
